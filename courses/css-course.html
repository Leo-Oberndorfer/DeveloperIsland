Introduction

CSS is used to style the html elements on a website. In this course you will learn how to add style and formatting to HTML documents, control the layout of a web page, create responsive designs, use selectors to target specific elements, work with web fonts, and use CSS preprocessors. CSS is an essential skill for any web developer or designer and is used in modern web development to create visually appealing and user-friendly websites.
